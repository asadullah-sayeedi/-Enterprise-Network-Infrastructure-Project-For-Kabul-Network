# Enterprise Network Infrastructure Project

This project is a simulated enterprise network designed using Cisco Packet Tracer. The network includes VLAN segmentation, Inter-VLAN Routing, DHCP configuration, Layer 3 switching, and web server integration.

## Features
- VLAN Configuration
- Inter-VLAN Routing
- DHCP Services
- Layer 2 & Layer 3 Switching
- Enterprise Topology Design
- Web Server with HTML/CSS
- Network Connectivity Testing
- Troubleshooting and Optimization

## Technologies Used
- Cisco Packet Tracer
- Cisco IOS CLI
- VLANs
- DHCP
- Routing & Switching
- HTML/CSS

## Objective
The purpose of this project is to practice real-world enterprise network design, improve troubleshooting skills, and understand scalable network infrastructure.
